# Mr.Snake
Mr. Snake is an HTML5 game based on Snake Classic.

## Aim
To feed our Hungry Mr.Snake with rats & earn the highest score. 

## Controls
Use arrow keys ( &uarr; &rarr; &darr; &larr; ) to control Mr.Snake.
